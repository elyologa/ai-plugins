# Changelog

All notable changes to the Claude Check Your Work Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-06

### Added

- Documentation evaluator agent for JSDoc and inline comment quality evaluation
- Health evaluator agent for structural health assessment relative to changes
- Unit test evaluator agent for test design quality review
- Skills: evaluate-documentation, evaluate-health, evaluate-unit-tests
- Skills: guide-on-situational-awareness, check-your-work
- Plugin manifest, README, and marketplace registration
