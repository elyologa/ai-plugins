## [1.0.0] - 2026-04-13

### Added

- `plan-implement-review` skill — end-to-end pipeline orchestrating requirements analysis, per-repo architecture planning, implementation with self-review, and multi-agent code review via Teams and task dependencies
- Reference implementation for composing marketplace agents (product-analyst, architect, software-engineer, security-engineer, code-reviewer) into complex multi-step workflows
